# Neon Tic-Tac-Toe

A dark neon-themed Tic-Tac-Toe game built with **HTML, CSS, and JavaScript**.

## How to Play

- Two players take turns clicking the cells.
- Player X and Player O alternate moves.
- First player to get three in a row (horizontally, vertically, or diagonally) wins.
- If all cells are filled without a winner, the game is a draw.
- Click the **Restart** button to play again.

## Features

- Dark neon aesthetic.
- Interactive and responsive.
- Works on desktop and mobile browsers.
- No backend required — fully static website.
